<h3>DESCRIÇÃO DO PROJETO</h3>

<P>1. Baixe os arquivos do projeto.</P>
<P>2. Abra um terminale escreva = pip install -r requirements.txt</P>
<P>3. Execute o arquivo app.py e entre no link.</P>
